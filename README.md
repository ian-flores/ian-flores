Senior Software Engineer at [Posit PBC](https://posit.co). I work on [team-operator](https://github.com/posit-dev/team-operator) and [ptd](https://github.com/posit-dev/ptd), which deploy and manage Posit's data science products — Connect, Workbench, and Package Manager — on Kubernetes. Day to day that means writing Go operators, wrangling Helm charts, debugging networking edge cases in customer clusters, and making sure data science teams can run R and Python workloads at scale without thinking about the infrastructure underneath.

Outside of work, I've been tinkering with an ecosystem of R packages for building LLM agent workflows: the [secure*](https://github.com/ian-flores?tab=repositories&q=secure&type=source) family covers sandboxed code execution, tool-call IPC, tracing, guardrails, and multi-agent orchestration. It's an experiment in bringing the kind of agent tooling that already exists in Python and TypeScript over to R.

Always down to collaborate on digital infrastructure for novel societal challenges — especially around education technology and open data.
